# Dummy data for cross-docking

Each subfolder is named `PDB_LIG` (e.g. `1G9V_RQ3`) and contains:

- `*_protein.pdb` — pocket / protein
- `*_ligand.sdf` — **reference** (native) ligand for that complex
- `query_<SOURCE_PDB>_<SOURCE_LIG>.sdf` — ligands from *other* complexes to dock into this pocket (cross-docking queries)

The subfolders are randomly extracted from the Astex Diverse set. 

## Cross-docking setup

Query SDFs were generated by `setup_crossdock_queries.py`: for each folder, the reference ligand (`*_ligand.sdf`) of every *other* folder is copied in as `query_<SOURCE>.sdf`. Re-run the script after adding new complexes:

```bash
python setup_crossdock_queries.py
```

## Using this for cross-docking sampling

- **Ligands to dock (queries):** use regex **`query_.*\.sdf`** so that only the cross-docking query SDFs are used as inputs.
- **Reference (for metrics):** use **`.*_ligand\.sdf`** as the native/reference ligand for the current complex (one reference per folder).

So for each `PDB_LIG` folder you dock all `query_*.sdf` molecules into that folder’s protein; the reference pose for evaluation is `PDB_LIG_ligand.sdf`.
